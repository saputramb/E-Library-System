# SKILL TEST

1. restore database PostgreSQL perpus.sql di pgAdmin
2. clone repository github
https://github.com/saputramb/E-Library-System.git
3. ubah file .env.example menjadi .env
4. setting .env, mulai dari HOST_DB, USERNAME_DB, PASSWORD_DB dan DATABASE_DB
5. buka terminal dan masuk ke directory backend lalu ketik pnpm install pada terminal
6. buka terminal dan masuk ke directory frontend lalu ketik pnpm install pada terminal
7. untuk menjalankan backend, masuk ke directory backend lalu ketik pnpm start
8. untuk menjalankan frontend, masuk ke directory frontend lalu ketik pnpm start
9. untuk masuk sebagai role admin masuk dengan menggunakan akun admin@gmail.com dengan password Admin123
10. untuk masuk sebagai role user masuk dengan menggunakan akun user@gmail.com dengan password User1234
